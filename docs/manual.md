# Tetorica Retro Player 取り扱い説明書

画像・動画・画面キャプチャを、レトロPC/CRT風のフィルターとローファイ音声エフェクトで加工しながら再生するプレイヤーです。
本書はアプリの使い方をまとめたユーザー向けマニュアルです（開発者向けの実装メモは `docs/issues/` を参照してください）。

---

## 1. 対応環境

| 配布形態 | 説明 |
|---|---|
| デスクトップアプリ（Tauri） | macOS / Windows / Linux。フル機能（後述の mDrop・ffmpeg 変換を含む）が使えます |
| Android アプリ（Tauri） | mDrop・ffmpeg 変換・仮想デバイス機能は無効。ファイル再生・フィルター・音声エフェクトは利用可能 |
| Chrome 拡張機能 | 現在開いているタブをキャプチャして加工、または他のWebページの動画・画像にフィルターを重ねる「Overlay」機能 |
| Web版（ブラウザ） | ドラッグ＆ドロップ・ファイル選択・画面キャプチャで再生。mDrop はネットワーク越しに稼働中のデスクトップアプリへアクセスする形でのみ利用可 |

---

## 2. メディアを読み込む

左上の ☰ (ハンバーガー) ボタンを押すとメニューが開きます。

- **ファイルを開く** — 動画・画像・音声・zip/pdf などをファイル選択ダイアログから読み込みます
- **フォルダを開く**（デスクトップのみ） — フォルダ内のメディアをプレイリストとして読み込みます
- **画面をキャプチャ** — OSの画面共有ダイアログでウィンドウ／画面全体を選び、そのままフィルターを通してプレビューします
- **マイク入力 / カメラ入力** — マイクの音声、またはカメラ映像を取り込みます
- **マイクデバイスを選ぶ** — 使用する入力デバイスを個別に指定します
- **Test プリセット** — カラーバー動画・カラーバー画像・ToneJS Lo-Fiデモ曲など、動作確認用のサンプルをワンタップで読み込めます（「More」でデモ曲一覧を展開）
- **言語** — 自動 / English / 日本語 を切り替えられます

ハンバーガーボタンを**長押し**すると、ツールバー全体を非表示にできます（配信・鑑賞用にUIを隠したい時向け）。非表示中は左上に小さな半透明ボタンが残るので、それを押すと元に戻ります。

このほか、メディアファイルを画面にドラッグ＆ドロップして読み込むこともできます。

### 対応ファイル形式

| 種別 | 直接再生できる形式 | ffmpeg変換（HLS）が必要な形式 |
|---|---|---|
| 動画 | mp4, m4v, webm, ogv, mov | avi, flv, mkv, wmv, ts, m2ts, mts, divx, xvid, rm, rmvb, 3gp, f4v, asf, vob, mpeg, mpg, m2v, mxf |
| 音声 | mp3, wav, ogg/oga, m4a, aac, flac, opus | — |
| 画像 | png, jpg/jpeg, webp, gif, svg, avif, heic/heif | — |
| その他 | mDrop 経由で共有された pdf / epub / zip(cbz) はブラウザ内でプレビュー可能（動画ではなく閲覧用） | — |

ffmpeg変換が必要な形式は、デスクトップアプリで ffmpeg が有効な場合のみ再生できます（詳細は後述の「mDrop」参照）。

## 3. 画面の基本構成

- **プレビュー枠**：フィルター適用後の映像が表示される領域
- **右上のツールバー**：Power・Hi-Res・明るさ・Fit width・Pin・Maximize・録画・⋯(More) ボタン
- **下部のコントロールパネル**：再生操作、Video / Audio 設定タブ、Reset、設定の保存・読み込み

---

## 4. 再生操作の基本

- **シークバー**：クリック／ドラッグで再生位置を移動
- **再生・一時停止**ボタン
- **音量**ボタン：短押しで音量スライダーを開閉、**長押し**でミュート切り替え
- **ループ**ボタン：押すたびに `1曲ループ → 自動で次へ → 全曲ループ → ループなし` の順に切り替わります
- **速度**（ゲージアイコン）：2x / 1.5x / 1.25x / 1x / 0.75x / 0.5x から選択
- **前後スキップ**ボタン：短押しで前後5秒シーク、**長押し**で前後のトラックに移動（プレイリスト再生時）

### キーボードショートカット

| キー | 動作 |
|---|---|
| Space または K | 再生・一時停止 |
| ← (左矢印) | 5秒戻る |
| → (右矢印) | 5秒進む |
| J | 10秒戻る |
| L | 10秒進む |

※ 入力欄にフォーカスがある間は無効になります。

## 5. 右上ツールバー

| ボタン | 動作 |
|---|---|
| **Power** | レトロフィルターのON/OFF |
| **Hi-Res**（絞りアイコン） | 描画解像度を 1x / 2x で切り替え |
| **明るさ**（−/％/+） | 40%〜200%の範囲で5%刻みに調整。**ウィンドウ幅が480px以上の時だけ**ここに表示され、それより狭い時は「More」メニュー内のスライダーに統合されます |
| **Fit width** | プレビューを横幅いっぱいに広げる／戻す |
| **Pin** | スクロールしてもプレビューを画面内に固定表示（Maximize中は使用不可） |
| **Maximize** | プレビューを全画面表示。もう一度押すか Esc で解除 |
| **録画**（●/■アイコン） | フィルター適用後の映像を録画。もう一度押すと停止して書き出し（幅360px未満では「More」メニューに移動） |

### 「⋯」More メニュー

- **Audio Optimize**：auto / chrome / safari — ブラウザごとの音声再生方式を手動指定（通常は auto でOK）
- **Latency**：interactive / balanced / playback — WebAudioの遅延優先度。変更はPowerのOFF→ON後に反映
- **QSV を使う**（Windowsのみ表示） — Intel Quick Sync対応環境でのffmpeg高速変換を有効化
- **HLS ffmpeg スロット数**（1〜8） — 同時に実行できる変換ジョブ数の上限
- **アラーム** — 時刻指定（HH:MM）＋ Set / Test / Clear ボタン、+1分 / +5分 / +10分のショートカット。指定時刻にメディア再生または通知音を鳴らします。バックグラウンド動作はブラウザ依存のため、他のウィンドウが前面にある場合など正常に動作しないことがあります
- **Flip H / Flip V** — 映像の左右・上下反転
- （狭い画面の場合）明るさスライダーもここに表示されます

## 6. Video 設定パネル

下部の「Video」タブを押すと開きます。長押しで Power と同じフィルターON/OFFを切り替えられます。

- **プリセット**：pc98_512 / fm77av / pc88va など、機種を模したレトロプリセットをタイルから選択
- **パレットモード**：RGBモード／グレースケール／モノクロなどの減色方式
- **モノクロトーン**（パレットがモノクロの時のみ）：グリーン・アンバー・アイスなどの単色調
- **出力サイズ**：幅・高さのスライダー、アスペクト比を元映像に合わせるチェックボックス
- **色数・ディザ・スムーズ**：減色の粒度、ベイヤーディザ、ピクセル化前のぼかし量
- **CRT/LCD効果**：曲面、走査線、走査線の明部フェード、ビネット、グロー、フォスファー、近接ノイズ
- **トーン圧縮**：明るさ・彩度のダイナミックレンジ圧縮（Amount/Low/High/Knee）
- **フォスファードット**（対応パレット時）：ドット形状、輝度、内部解像度2倍化などの詳細設定
- **アニメ/トゥーン**：ポスタライズ段数、輪郭線の強さ・検出しきい値

各項目はスライダーやチェックボックスで、動かすとリアルタイムにプレビューへ反映されます。まずはプリセットを選んでから微調整するのがおすすめです。

---

## 7. Audio 設定パネル

下部の「Audio」タブを押すと開きます。長押しで音声エフェクトのON/OFFを切り替えられます。

- **Effects on/off**：音声エフェクト全体のON/OFF（長押しでFFTスペクトラム表示のON/OFFも可能）
- **Noise on/off**：ノイズ効果のON/OFF
- **プリセット**：Classic / Lofi / 8-bit / VHS などをタップで一括適用
- **エフェクト**：Lo-Fi量、ラジオトーン、ビットクラッシュ、サンプルレート低下、ノイズ低減、ワウ・フラッター、Bass/Mid/Trebleの3バンドEQ
- **空間系**：ディレイ、リバーブ、コーラス
- **ダイナミクス**：テープサチュレーション、コンプレッサー
- **入出力**：音量、ステレオ幅、スモールルーム感、出力/入力ゲイン
- **ノイズ**：サーフェスヒス（レコード盤のヒスノイズ）、ヴァイナルダスト（プチノイズ）、暖かみ、エア感、プレゼンス

---

## 8. Reset / Native再生モード

下部の「Reset」ボタン：

- **短押し**：Video/Audioの設定をすべて初期値に戻す
- **長押し（0.6秒）**：Native再生モードに切り替え。ボタン表示が「Native」に変わります

Native再生モードは、レトロフィルターを適用せずOSネイティブの動画再生に切り替えるモードです。特定の環境（Windows上でのHLSストリーミング再生など）でフィルター経由の再生が不安定な場合の代替手段として使えます。

---

## 9. 設定の保存・読み込み

下部コントロールパネル右端のアイコンから：

- **保存**（フロッピーアイコン）：Video/Audio/UI設定をまとめて `retro-YYYYMMDD-<timestamp>.retro.json` として書き出します
- **読み込み**（フォルダアイコン）：`.retro.json` をファイル選択またはドラッグ＆ドロップで読み込み、即座に設定へ反映します

お気に入りの見た目・音を作ったら保存しておき、別のメディアを開いたときに読み込んで使い回せます。

## 10. mDrop（別端末からのファイル転送）

**デスクトップアプリのみ**の機能です（Android / Chrome拡張 / Web版では使えません）。同じネットワーク内の別端末（スマホなど）からブラウザ経由でファイルを送り込めます。

画面右上に「mDrop」ピルボタン（Wifiアイコン）が表示されます。

- **クリック**：転送用サーバーのON/OFFを切り替え（起動時は自動でON）
- **長押し**：「共有モード」のON/OFF。ONにすると画面に `IP:ポート`（例: `192.168.1.5:7878`）が表示され、他の端末のブラウザからその URL を開けるようになります
- 共有モードで開いた mDrop Web UI からファイルをアップロードすると、即座にこのアプリのプレビューに反映されます
- 送られてきた PDF / EPUB / ZIP(cbz) はブラウザ内でそのままプレビューできます

隣に「ffmpeg」ピルボタン（波アイコン）も表示される場合があります。ONにすると、ブラウザが直接再生できない動画形式（mkv/aviなど）をHLSストリーミングとして配信できるようになります（ffmpegがインストールされた環境が必要）。

## 11. Chrome 拡張機能

`chrome://extensions` から読み込む、または Chrome ウェブストアからインストールして使う簡易版です。拡張機能アイコンをクリックするとポップアップが開きます。

- **現在のタブをキャプチャ**：表示中のタブの映像をそのまま取り込んでフィルター加工
- **ビューアーを表示**：専用タブでキャプチャ結果を表示
- **Overlay（実験的機能）**：他のWebページ上の動画・画像に直接レトロフィルターを重ねて表示
- ポップアップ内には本体アプリ同様の **Video** / **Audio** タブに加え、**Overlay** の詳細設定、簡易版の **Alarm** タブがあります
- 設定は Chrome 拡張機能のストレージに保存されます（本体アプリの `.retro.json` とは別管理）

## 12. こんな時は

- **音が出ない / 「Touch & Play」から進まない**：Safari など一部ブラウザはユーザー操作なしに音声再生を開始できません。表示されているボタンを押して再生を開始してください
- **「読み込み中」が続く / フォーマット非対応と表示される**：ffmpeg変換が必要な形式の可能性があります。デスクトップアプリで ffmpeg が有効か確認してください
- **再生がカクつく・止まる**：Reset ボタンの長押しで Native再生モードに切り替えると改善することがあります
- **Windows で映像は動くのに音が出ない／エフェクトが効かない**：HLS再生時にWeb Audioを経由しない既知の制限です。詳しくは `docs/issues/windows-tauri-hls-webaudio-bypass.md` を参照してください
- そのほかブラウザ・OS固有の既知の癖は `docs/issues/knowhow-os-browser-for-retroplayer.md` に一覧をまとめています
