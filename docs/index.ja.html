Tetorica Retro Player

画像・動画・画面キャプチャを、同じレトロCRT/PC風フィルターとローファイ音声エフェクトで加工しながら再生できます。 複数ファイルをプレイリストにしたり、mDropでスマホからファイルを転送したり、アラームを設定したり、 加工後の映像をそのまま録画したりも可能。macOS / Windows / Linux 向けデスクトップアプリ、 Androidアプリ、Chrome拡張機能、そして下のブラウザデモから使えます。

PWAデモを開く 取扱説明書を見る Chrome拡張機能を開く ソースコードを見る itch.io で見る（デスクトップ・Android）
入力 画像・動画ファイルのドラッグ＆ドロップ、または画面・ウィンドウのキャプチャ。
出力 減色、モノクロ調、ディザリング、走査線、CRT風の雰囲気。

デモで試せること

写真・動画クリップ・キャプチャしたアプリウィンドウから始めて、PC-98風の色数制限、 モノクロ端末風の見た目、CRT/フォスファー仕上げ、ローファイ音声エフェクトを切り替えて試せます。 複数ファイルをプレイリストとして読み込んだり、お気に入りの設定をファイルとして保存したりも可能。 デモで触れていない部分は取扱説明書にまとめています。

レトロ映像フィルター パレットプリセット、モノクロ調、ディザリング、CRTの曲面・走査線・フォスファードットなどをリアルタイムに調整できます。
ローファイ音声エフェクト ビットクラッシュ、ワウ・フラッター、テープサチュレーション、ヴァイナルノイズ、リバーブ、3バンドEQでレトロな見た目に音も合わせられます。
mDrop ＆ プレイリスト 同じネットワーク内のスマホからファイルを送り込んだり（デスクトップアプリ）、複数ファイルをループ再生のプレイリストにまとめたりできます。

デスクトップの Chromium 系ブラウザで最も快適に動作します。画面キャプチャの可否はブラウザの権限設定とプラットフォームに依存します。 機能の全体像やプラットフォームごとの違いは取扱説明書を参照してください。
