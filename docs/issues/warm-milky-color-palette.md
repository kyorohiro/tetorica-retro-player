# Warm Milky Color Palette

アプリ全体に適用したウォームミルク系の配色について。

## 特徴

彩度を落とすだけでなく、**色相をウォーム方向に振る**のがポイント。

```
変更前（クールブルー系）
#eceef4  → H:228° S:14% L:93%
#f8f9fc  → H:228° S:8%  L:98%

変更後（ウォームミルク系）
#eae6df  → H:37°  S:16% L:88%
#f5f1ea  → H:37°  S:22% L:95%
```

彩度はどちらも低い（10〜25%）。違いは色相が正反対：

- クールグレー: H ≈ 220〜230°（青方向）
- ウォームミルク: H ≈ 30〜45°（黄橙方向）

## 作り方

```
白から青を引く          → クールグレー
白から青を引いて黄みを足す → ミルク/クリーム
```

Figma での手順：
1. 白 (#ffffff) から始める
2. H を 35〜45 に設定
3. S を 15〜25% に抑える
4. L を 85〜95% で調整

## 実際の色マッピング

| 用途 | 変更前 | 変更後 |
|------|--------|--------|
| ページ背景 | `#eceef4` | `#eae6df` |
| カード背景 | `#f8f9fc` | `#f5f1ea` |
| サブセクション背景 | `#f0f2f7` | `#ede8e2` |
| ホバー | `#e2e5ee` | `#e2ddd5` |
| メインボーダー | `#c8cede` | `#cac0b2` |
| サブボーダー | `#b8c0d4` | `#bcb4a6` |
| ミュートテキスト | `#7a88a8` | `#7a7268` |
| ライトボーダー | `#a8b4c8` | `#a8a29a` |

body にはウォームなラジアルグラデーションも追加：

```css
background:
  radial-gradient(ellipse at 20% 0%, rgba(240, 232, 218, 0.7) 0%, transparent 55%),
  radial-gradient(ellipse at 80% 100%, rgba(228, 222, 210, 0.5) 0%, transparent 50%),
  #e8e3dc;
```

## 背景・由来

- au INFOBAR (2003, Naoto Fukasawa) の乳白色×赤の配色が有名
- ただしあの配色自体、60〜80年代の西欧ポップデザイン（フィフティーズのアメリカンダイナー、ブリキ玩具）の影響
- 古い雑誌の紙が黄ばむ方向が H:35〜45° で、このパレットと一致 → レトロ感と相性がいい理由
- 「ほぼ白だけど白じゃない」という極端に彩度の低いゾーンで遊んでいる

## 注意点

`backdrop-blur-sm`（backdrop-filter）を外側の section に付けると、
子孫の `position: fixed` 要素（ピン・最大化）の containing block が viewport でなく
その section になってしまい、位置がズレる。ミルク感は半透明背景だけで十分出る。
